# Kanban_Js

Projeto de um **Kanban de Tarefas** desenvolvido com **HTML, CSS e JavaScript**, com o objetivo de organizar atividades de forma visual, simples e prática.

## Sobre o projeto

<img width="1399" height="727" alt="Captura de tela 2026-03-28 000107" src="https://github.com/user-attachments/assets/16645fa8-d7ab-4494-b71d-0062ad38e77b" />


O **Kanban** é um método de organização visual usado para acompanhar o fluxo de tarefas por etapas. Ele ajuda a entender com clareza:

- o que precisa ser feito
- o que está em andamento
- o que já foi concluído

Neste projeto, as tarefas são organizadas em três colunas:

- **A Fazer**
- **Fazendo**
- **Concluído**

A aplicação foi desenvolvida com foco em prática de lógica de programação, manipulação do DOM e construção de interfaces interativas com JavaScript.

## Funcionalidades

- Adicionar tarefas
- Editar tarefas
- Excluir tarefas
- Mover tarefas entre etapas
- Alterar a cor dos cards conforme o status
- Salvar tarefas no navegador com **localStorage**

## Tecnologias utilizadas

- **HTML**
- **CSS**
- **JavaScript**
- **localStorage**

## Objetivo do projeto

Este projeto foi criado para praticar conceitos importantes do desenvolvimento front-end, como:

- estruturação de páginas com HTML
- estilização com CSS
- interatividade com JavaScript
- manipulação de elementos na tela
- armazenamento local de dados

Além disso, o projeto também faz parte do meu processo de construção de portfólio na área de desenvolvimento web.
